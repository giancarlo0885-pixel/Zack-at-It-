# V19 Autonomous Intelligence

V19 adds an evidence-backed command layer that summarizes the platform into a single operating posture: DEPLOY CAPITAL, SELECTIVE OFFENSE, WATCH AND PREPARE, PRESERVE CAPITAL, or SYSTEM DEFENSE.

The posture uses current opportunity quality, modeled probability, net expected value, combined portfolio cash, active risk warnings, worker status, and provider readiness. Missing evidence lowers readiness; it is never fabricated.
